### 单因素方差分析（One-way Anova）与T检验的用途很类似：
1. 都需要样本数据符合正态分布；
2. 都是检验样本均值是否有显著差异；
3. 都需要满足方差齐性要求（方差不齐没有可比性）；
4. 都是针对单因子进行分析。
